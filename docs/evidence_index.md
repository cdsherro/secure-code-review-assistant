# Evidence Index

This folder contains the baseline evidence for Checkpoint 2.

## Files included

- `before_measurement.md`
  Summary of the initial baseline results.

- `known_issue_list.md`
  The issue list I used to compare against the baseline LLM review.

- `llm_baseline_review.md`
  Notes from the first-pass LLM review.

- `environment_setup.txt`
  Local Python and Flask setup notes.

## Notes

The goal of this checkpoint was to create a real starting point for the project. These files represent the before state of the LLM review process before prompt improvements.

No real secrets or private data are included in this repo.
